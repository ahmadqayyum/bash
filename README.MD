

# Install minikube in Ubuntu

![Image result for minikube](https://cdn-images-1.medium.com/max/2400/1*RPlkI7b6Ek85uyaQ3qPS8w.jpeg)


## Requirement 
  - Ubuntu 16.04 Or Ubuntu 18.04
  - Oracle Virtual Box
  - Super privilege 

## Run Script
```
chmod +x minikube.sh
./minikube.sh
```
